jquery基础

发表于 2016-02-01

jquery

js框架库:

  • 把一些常用的函数进行封装,封装成一个js文件.这个文件就是库,称为:javascript框架库,函数库
  • 常见的框架库：Prototype、YUI(网络反响一般)、Dojo、ExtJS、jQuery

  • dom中的缺点:代码容错性差,window.onload只能写一次,兼容问题特别多

  • JQuery:免费,插件丰富,开源

  • JQuery特点:写的少，做的多
  • $(‘’);页面要统一,如果页面中有多个地方是用单引号,建议都用单引号的方式
  • $(“”);

    jquery使用：

  • 第一个问题.写jq代码,要先引入JQuery的问题,
  • 第二个问题:不要在引入jquery的script标签中写任何的代码
  • 第三个问题:如果编程过程中,发现代码一点都没有问题,看看是不是出现了第二个问题
  • jquery的代码一定要写注释,代码结束一定要加分号

    顶级对象

  • dom中的顶级的对象document
  • bom中的顶级的对象window
  • JQuery中的顶级对象:$—jQuery

    加载页面有几种方式:

  • dom 加载页面 window.onload=function(){};
  • JQuery中加载页面方式:
    • 第一种:页面中所有的标签.图片.外部引入的文件都加载完毕再执行
      • $(window).load(function(){});
    • 第二种:页面中的dom元素加载完毕之后就执行
      • $(document).ready(function(){});比第一种快
    • 第三种:和第二种一样,开发的时候
      • $(function(){});
    • 第四种:变种
      • jQuery(function(){});

        jquery书写注意

  • jquery代码中$都可以替换成jQuery单词
  • 在jquery中写代码的时候,把dom中事件的on去掉,直接.后面的事件方法的名字就可以了
    • $(“#btn”).click(匿名函数);

$对象和dom对象互转

  • 原生js(最基本的js代码)
  • array–js内置对象
  • date–js内对象
  • RegExp–正则
  • dom对象—标签—通过document获取的对象
  • $对象—只要是对象被$包裹着就是$对象
  • jquery对象转dom—->$(“#btn”)[0]
  • jquery对象转dom—->$(“#btn”).get(0);
  • dom对象转jquery对象—>$(dom对象);

  • $(this) var obj=$(this); obj.innerHTML

  • dom对象转jquery对象编程更方便

  • jquery对象转dom对象是为了解决兼容问题(写原生的js代码)

    识别对象

  • 如何识别这个对象是jquery对象:
  • 可以通过.语法获取方法进行使用,如果不能调用就不是jquery对象
  • 如何识别这个对象是dom对象
  • 可以通过.语法获取属性进行使用,能用就是dom对象
  • this是dom对象
  • jquery–this要转jquery对象—经常用
  • $(this)表示的是jquery对象
  • dom中获取value的值 .value
  • jquery中用的是.val();
  • .val()如果括号中什么也不写表示的是获取value的值
  • .val()如果括号中写了一个字符串,表示的是设置value的值

    选择器

  • 选择器:作用:用来获取需要的元素
  • dom中通过什么方式获取元素的呢?
  • document.getElementById———一个对象
  • document.getElementsByTagName—-多个对象
  • document.getElementsByClassName—多个对象

  • jquery中获取元素使用选择器(很多–一大坨)

    重点的选择器:

    • 三个:id选择器,标签选择器,类选择器
    • id选择器: 是在$(“#btn”)在双引号中写一个#id的值
      • dom中的事件到了jquery中去掉on,直接写事件的名字加小括号
      • 此时这个事件就是一个方法了,然后在方法中写一个匿名函数即可
    • 标签选择器:在$(“标签的名字”),获取的是多个标签
      • .text()方法,如果在里面写了一个参数(字符串类型),就是设置这个标签的innerText内容
      • .text()啥也没写,获取标签的innerText内容

    • 类选择器:在$(“.类样式的名字”)表示获取在这个页面上所有应用了这个类样式的标签

      • .css(),在css()方法中可以写一个属性，逗号隔开，这个属性对应的值
      • 如果是多个属性和多个值可以使用大括号的方式,键值对的方式
      • $(“.cls”).css(“backgroundColor”,”green”);

    • .html()相当于innerHTML

    • .html(“标签加文字内容”)

    • 标签加类选择器

      • 在$(“标签的名字.类样式的名字”)
      • 解释:获取所有的span标签,是应用了cls这个类样式的span标签
    • 多条件选择器:在$(“多个条件”),每个条件都是由逗号隔开
    • 子代选择器:当前的标签里面的直接的子元素
      • $(“父级的条件>子级的条件”)

    • 后代选择器:当前的标签里面的子元素的子元素,仔仔元素()

        • 获取当前元素的后面的第一个兄弟元素
      • ~ 获取当前元素的后面的所有的兄弟元素
      • div~span

    • 奇数的选择器

      • $(“#u1>li:odd”).css(“backgroundColor”,”red”);
    • 偶数的选择器
      • $(“#u1>li:even”).css(“backgroundColor”,”blue”);
    • 索引选择器
      • :eq(索引)获取对应索引的那个标签
      • :gt(索引)获取大于那个索引的所有的标签
      • :lt(索引)获取小于那个索引的所有的标签
    • .children()获取当前元素的子元素,在括号中写标签的名字那么获取的就是子元素中的某个标签
    • .show()就是显示出来 .hide()就是隐藏
    • .parent()方法是获取当前元素的父级元素
    • .find(),方法中写一个参数.就是标签的名字,获取到的就是这个标签名所有的标签
    • .siblings()获取当前元素的所有的兄弟元素
      • 如果在括号中写了标签的名字，那就是获取指定的兄弟标签
