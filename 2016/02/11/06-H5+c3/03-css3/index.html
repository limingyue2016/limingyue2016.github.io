css3第二天

背景

  • background-size:

    设置背景的大小,通过这个属性可以使用多张图片作为背景

    • cover(等比例放大，直到两边都顶格，如果图片变大，超出的会切掉)
    • contain(等比例放大，某一边顶格，立刻停止)

  • background-origin:

    设置的是背景图片的开始位置

    • border-box
    • content-box
    • padding-box

  • background-repeat

    在原来只能设置平铺,不平铺的基础上,增加了两个属性,让我们在设置背景图时更为灵活

    • / 自动缩放 直到填满 /
    • background-repeat:round;
    • / 两个方法都缩放,直到某一边充满容器为止 /
    • background-repeat:space;

  • border-image:

    • border这个属性必须设置
    • border-image-source:url(“”)
    • border-image-slice: x y 没有单位
    • border-image-width：
    • border-image-repeat:stretch/round/repeat
      • round 可以保证图片完整性
      • repeat 只是简单的重复 可能会有半个图片显示

  • background 支持设置多个背景图，最后放一个背景颜色

    • background：source repeat position /size origin(注意：position要有才能写size)

body 的注意事项：

  • 默认没有高度 需要设置html，body{height：100%；}
  • body{margin:0;box-sizing:border-box;} //如果设置了border

过渡 transition

  • transition-property:
  • transition-duration:
  • transition-delay:
  • transition-timing-function:ease/linear(线性匀速)/steps()（步骤动画）
    • 贝塞尔曲线动画 cubic-bezier()
  • transition:property duration delay timing-function，…

2D转换 transform

  • transform-translate(x,y)//移动元素不会影响其他元素
  • transform-rotate(n deg)//旋转 %也可以
  • transform-scale() //缩放 默认是1
  • transform-skew() //扭曲 如果两个轴扭曲角度一致，元素会消失
  • transform-origin:具体位置单词 /具体坐标
  • transform:translate() rotate() scale()
  • 注意：先走还是先旋转 效果不一样的
