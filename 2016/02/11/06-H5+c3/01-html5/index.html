新表单属性

  • placeholder(占位符)
    • 输入元素内默认显示一些提示信息,当用户输入之后自动消失
  • autofocus(获取焦点)
    • 当某一页有很多个可供输入的元素时,用户需要使用鼠标点选元素进行输入,为了提升用户体验,我们可以通过autofocus属性来指定页面中默认选中的元素

Dom语法

  • document.querySelector()

    可以传入的选择器 还有CSS3的选择器 获取满足条件的第一个元素 从上往下 查找

  • document.querySelectorAll()

    返回数组

  • classList

    这个元素返回的是dom元素的class 列表,这个属性本身是只读的，但是我们可以通过几个方法去操作元素的class

    • 可用方法：
      • add ： 添加一个 class 到元素的 class 列表中
      • remove ： 从元素的 class 列表中移除一个 class
      • toggle ： 切换一个 class 是否存在于一个元素的 class 列表中
      • contains ： 查询一个指定的 class 是否存在于元素的 class 列表中

data-添加属性

  • 语法:
    • data-:这个是固定格式
    • data-xxx:xxx是自定义的值
  • 注意:
    • 属性名不应该包含任何大写字母，并且在前缀 “data-“ 之后必须有至少一个字符
    • 属性值可以是任意字符串
    • 不要使用纯数字,特殊字符等作为自定义值
    • 如果data之后使用-连接多个值,取值时,需要使用驼峰命名
  • 获取：
    • 对象.dataset 返回是自定义属性的对象，可以.出想要的属性值或者 也可以 对象.dataset[“属性名”]
    • 如果定义的时候使用了大写字母获取的时候 会变成小写,我们需要通过小写字母来获取
    • 如果 我们使用 多个 - 进行分割会去掉横线,横线之后的 首字母变为大写
