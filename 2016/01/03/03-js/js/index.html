js基础

简介

  1. 运行在客户端的脚本编程语言
  2. 三部分组成
  • ECMAScript
  • DOM
    • BOM
  1. 特点：解释执行语言，基于对象，具有继承性，封装性，多肽特点
  2. 书写位置类似css

    数据类型

  • 简单数据类型：string,number,boolean,undefined
  • 复杂数据类型：array,object
  • 补充：数据类型是确认变量存储数据的位置的
  • 判断数据类型使用typeof关键字

    运算符

  • 算数运算符
  • 比较运算符
  • 逻辑运算符

    Math方法

  • .ceil() .floor() .random() .round() .pow() .abs()

    数据类型转换

  1. string->number
    • Number() parseInt() parseFloat()
  2. number->string
    • String() 变量.toString()
  3. 转boolean
    • 0 null undefined NaN ‘’是false
  4. boolean->number
    • Number()

      判断语句

  • if语句
  • 三元运算符

    循环语句

  • for（）
  • do（） while
  • while

    break和continue

  • break 遇到后循环程序会立即结束
  • continue 遇到后结束本次循环，开始下一次循环,continue后的代码不执行，整个循环还在执行

    补充

  • js对小数不精确，不能用“==”来比较
  • number.toFixed 保留小数位数，0-20
