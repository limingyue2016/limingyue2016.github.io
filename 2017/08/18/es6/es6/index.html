let和const命令

  1. for循环，设置环境变量的是一个父级作用域，循环体是一个子集作用域
  2. 不存在变量提升
  3. 只要块级作用域内存在let命令，它所声明的变量就“绑定”（binding）这个区域，不再受外部的影响。也不让他继承全局变量的值
  4. 避免在块级作用域内声明函数，非要的话用函数表达式（前提是在大括号内才可以），不用函数声明语句。
  5. const的本质是保证变量指向的内存地址不变

变量的解构赋值

  • 数组，对象
  • 解构:允许按照一定模式，从数组和对象中提取值，对变量进行赋值,如果解构不成功，变量的值就等于undefined
  • 默认值：注，es6内部使用（===），判断一个位置是否有值，所以一个数据成员不严格等于underfind，默认是不会生效的。
  • 解构赋值的用途
    • 交换变量
    • 从函数返回多个值，可直接赋值给变量
    • 函数参数的定义
    • 提取json数据
    • 加载指定模块，提高效率

字符串扩展

  • 多了一些方法 多数是解决之前方法的局限性
  • 模板字符串
    • 支持换行
    • 使用 ${} 引用变量，可以直接写表达式

函数的扩展

  1. 函数参数的默认值 简洁自然 注：参数变量是默认就声明的
  2. length属性，当参数有默认值时候，失效；如果设置了默认值的参数不是尾参数，那么length属性也不再计入后面的参数了
  3. rest 参数（形式为…values）…是rest标记
    用于获取函数的多余参数，rest 参数搭配的变量是一个数组，拿到所有剩余参数放到一个数组中

数组扩展

  • 展开运算符 它好比 rest 参数的逆运算，将一个数组转为用逗号分隔的参数序列。
    console.log(...[1, 2, 3]) // => console.log(1, 2, 3)

箭头函数

  • 作用 简化回调函数，匿名函数中使用广泛
  • 不适用于构造函数中
  • 如果需要一个多次使用的函数，还是通过 function 去定义
  • 注意：函数体内this的对象，就是定义时所在的对象，而不是使用时所在的对象
