JavaScript Standard Style

参考来源

细则

  • 使用两个空格缩进

  • 除需要转义的情况外，字符串统一使用单引号

  • 不要使用未定义的变量

  • 关键字后边加空格,函数名和括号加空格，,后边加空格，对象键值对的：后加空格

  • 函数调用时 标识符与括号不留间隔

  • 模板字符串中变量前后不加空格

  • 使用浏览器全局变量时加上window前缀

  • 不允许有连续多行空行
  • 每个var关键字单独声明一个变量

  • 单行代码块两边加空格

  • 对于变量和函数名统一使用驼峰命名法

  • 文件末尾留意空行
  • 无参的构造函数调用时要带上括号

  • 不要省去小数点前面的0

  • 不要混合使用空格与制表符作为缩进

  • return 语句中的赋值必需有括号包裹

  • 注释首尾留空格

关于分号

  • 不要使用分号
  • 不要使用 (, [, or `等作为一行的开始。
