数组和函数

数组

  • 定义
    • var ary=[1,2,3]
    • var ary= new Array()
      • 参数位置一个数值时为数组长度，多个数值时为数组中的元素
  • 遍历

函数

  • 定义：可以重复执行的代码块
    • function 名字（）{程序}
  • 参数
    • 形参 实参
  • 返回值
    1. 如果函数没有使用 return ，那么函数有默认的返回值：undefined
    2. 如果函数使用 return，那么跟再return后面的值，就成了函数的返回值
    3. 如果函数使用 return，但是return后面没有任何值，那么函数的返回值也是：undefined
    4. 函数使用return后，这个函数会在执行完 return 语句之后停止并立即退出，也就是说return后面的所有其他代码都不会再执行。

      函数名 函数体 函数加载问题

  • 函数名 == 整个函数
  • JS加载的时候，只加载函数名，不加载函数体。所以如果想使用内部的成员变量，需要调用函数。

    变量和作用域

  • 隐式全局变量

变量声明提升（预解析）

  • 函数中，定义变量在使用变量之后。（先提升变量，后提升函数）
  • 只提升变量名，不提升变量值，容易出现undefined。计算后形成NaN
  • 函数是所有内容全部提升。（function直接定义的）

    函数高级

  • 匿名函数 匿名函数就是没有名字的函数
  • 递归函数 自己调用自己
  • 回调函数 函数作为参数
  • 闭包 函数作为返回值

      文本属性

补充

  • 1em代表一个文字的大小

    标签显示方式及转换

  • 块级元素
    • 独占一行；可以设置宽高；没有设置宽高情况下，宽度随父级元素，高度是内容的高度
  • 行内元素
    • 在一行显示，不能设置宽高
  • 行内块元素
    • 代表 img和input；元素在一行显示；可以设置宽高
  • 元素模式转换
    • display:block/inline-block/inline

      css特性

  • 层叠性
    • 同一标签，权重相同，样式发生冲突时，最后定义的样式会将前面的样式覆盖
  • 继承性
    • 给父元素设置样式时，子元素在没有默认样式的情况下会受父元素样式的影响
    • 可以继承的属性：color,与文本相关的属性，text-align
    • 高度宽度不能继承
    • 继承特殊性：a标签不会受到父元素颜色的影响；标题标签不会受父元素文字大小影响，要与父元素的文字大小进行乘积运算
  • 优先级
    • 标签<class<id<行内样式<！important
    • 继承的优先级为0
    • 权重可以叠加

      伪类

  • a:link 推荐使用a
  • a:visited 只能设与颜色相关的属性
  • a:hover 用的多
  • a:active

    盒子模型

  • border
    • 属性 width style color
    • style的默认值是none，solid dashed dotted
    • outline-style: none;去轮廓线
    • border-collapse:collapse;只在表格里实现合并边框
  • padding
    • 在块级元素中，如果默认子元素没有设置宽度，给子元素设置padding，不会影响子元素大小
  • margin
    • 1.垂直外边距合并，水平方向没有
    • 2.垂直外边距塌陷
      • a给父元素设置边框
      • b给父元素设置overflow: hidden;

        行高

  • 行高=文字大小+上下间距
  • 可以实现让文字垂直居中,只能让文字垂直居中，行高值和容器的高度一致
  • 特点：文字大小、字体影响行高，如果设置好行高的话跟文字大小就没关系了,行高只能影响文字，不影响标签

    补充

  1. 浏览器中默认文字大小16px，行高18px
  2. 点击鼠标获得焦点 label for=”id”

    浮动及清除浮动

  • flow:left|right
  • 特点：
    1. 设置后不占原来的位置，托标（行内元素浮动可设置宽高，因为脱标）
    2. 可以在一行显示
    3. 实现模式转换
    4. 会影响浮动元素后边的元素
    5. 想让块级元素在一行显示，需要给所有的块级元素设置浮动
  • 作用：
    1. 布局
    2. 制作网页导航
    3. 实现文字图片环绕 文字不会被浮动元素压着

  • 清除浮动

    • 什么时候清除浮动
      • 父元素没有高度并且所有子元素都设置了浮动

    • 清除浮动三种方法：

      1. 使用.clear:both(

        content=“”；
        heitht：0；
        line-height:0;
        clear:both;
        display:block;
        visibility:hidden;
        ```

overflow的用法

  • 默认是visible
  • hidden
  • scroll
  • auto

    定位

  1. 静态定位
    • 标准流下的效果
  2. 绝对定位（看脸型）
    • 脱标不占位置 实现模式转换，有盒子压盒子时候使用
  3. 相对定位（自恋型）
    • 占位置，子绝父相
  4. 固定定位
    • 以body为准，脱标不占位置，实现模式转换

      绝对定位居中问题

  • left：50%；父元素的一半，margin-left：-自己的一半

    z-index

  • 只有定位的元素（除static）才有层级关系
  • 默认值auto（0）
    • z-index相同，后来居上
    • 如果父元素设置了，值越大越靠前

      网页布局顺序

  • 标准流 浮动 定位

    vertical-align

  • 行内块元素设置垂直居中的属性
  • 默认值 baseline，top，middle，bottom
    • 图片垂直居中 设置行高和middle值

      元素隐藏

  • overflow:hidden
  • display:none;display:block; 隐藏元素不占位置
  • visibility:hidden

    logo内容移除

  • text-indent:-2000px;
  • logo一般是给a设置背景图片，a要模式转换

    精灵图

  • 一种特殊的背景图片
  • 设置bg-img，使用fw的“打开”方式，找到横纵坐标
