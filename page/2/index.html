表单标签中的value属性值设置和获取

  • 文本框,按钮,复选框,单选框,文本域,下拉框都可以使用.val()方法获取value属性值
  • 通过$(“select”).val(值);可以设置某个选项选中

    attr()方法可以设置或者获取标签自带属性和自定义属性的值

    $("chekcbox").attr("checked"); undefined
  • prop()方法可以设置或者获取标签的属性值,而且可以获取复选框checked的true或者false值

    获取宽高

  • 通过 .width()方法和.height()方法可以获取元素的宽和高,而且是数字类型
  • 通过 .width(数字)和.height(数字)可以设置元素的宽和高
  • 通过 .scrollLeft()和.scrollTop()方法可以获取卷曲出去的距离
  • 通过.offset().left和.offset().top可以获取元素的横纵坐标
  • 通过.offset({“left”:值,”top”:值})可以设置元素的横纵坐标

绑定事件

  • 第一种:
    • 对象.事件名字(匿名函数);
    • 举例子: $(“#btn”).click(function(){});
  • 第二种:
    • 对象.bind(事件的类型,匿名函数);
    • 举例子:$(“#btn”).bind(“click”,function(){});
    • $(“#btn”).bind({“事件”:匿名函数,”事件”:匿名函数});
  • 第三种:
    • 父级元素对象.delegate(子级元素对象,事件的类型,匿名函数);
    • 举例子:$(“div”).delegate(“p”,”click”,function(){})
    • delegate()方法内部调用的是on()这个方法实现的
  • 第四种:
    • 父级元素对象.on(事件的类型,子级元素对象,匿名函数);
    • 举例子:$(“div”).on(“click”,”p”,function(){});
    • 但是.on()方法也可以这么写
    • 对象.on(事件的类型,匿名函数);
    • 举例子:$(“#btn”).on(“click”,function(){});

解除绑定事件:

  • bind()绑定用unbind()解除
  • delegate()绑定用undelegate()解除
  • on()绑定用off()解除

    触发事件

  1. 对象.事件名();
  2. 对象.trigger(“事件名”);
    • 前两种触发事件会触发浏览器的默认事件,下面的不会
  3. 对象.triggetHanele(“事件名”);

事件对象可以调用keycode属性获取键的值

在jquery中阻止事件冒泡

  • return false可以取消浏览器的默认事件，阻止事件冒泡

  • 定义：可以重复执行的代码块
    • function 名字（）{程序}
  • 参数
    • 形参 实参
  • 返回值
    1. 如果函数没有使用 return ，那么函数有默认的返回值：undefined
    2. 如果函数使用 return，那么跟再return后面的值，就成了函数的返回值
    3. 如果函数使用 return，但是return后面没有任何值，那么函数的返回值也是：undefined
    4. 函数使用return后，这个函数会在执行完 return 语句之后停止并立即退出，也就是说return后面的所有其他代码都不会再执行。

      函数名 函数体 函数加载问题

  • 函数名 == 整个函数
  • JS加载的时候，只加载函数名，不加载函数体。所以如果想使用内部的成员变量，需要调用函数。

  • 只提升变量名，不提升变量值，容易出现undefined。计算后形成NaN
  • 函数是所有内容全部提升。（function直接定义的）

    函数高级

    • outline-style: none;去轮廓线
    • border-collapse:collapse;只在表格里实现合并边框
  • padding
    • 在块级元素中，如果默认子元素没有设置宽度，给子元素设置padding，不会影响子元素大小
  • margin
    • 1.垂直外边距合并，水平方向没有
    • 2.垂直外边距塌陷
      • a给父元素设置边框
      • b给父元素设置overflow: hidden;

        行高

  • 行高=文字大小+上下间距
  • 可以实现让文字垂直居中,只能让文字垂直居中，行高值和容器的高度一致
  • 特点：文字大小、字体影响行高，如果设置好行高的话跟文字大小就没关系了,行高只能影响文字，不影响标签

    补充

  1. 浏览器中默认文字大小16px，行高18px
  2. 点击鼠标获得焦点 label for=”id”

  • flow:left|right
  • 特点：
    1. 设置后不占原来的位置，托标（行内元素浮动可设置宽高，因为脱标）
    2. 可以在一行显示
    3. 实现模式转换
    4. 会影响浮动元素后边的元素
    5. 想让块级元素在一行显示，需要给所有的块级元素设置浮动
  • 作用：
    1. 布局
    2. 制作网页导航
    3. 实现文字图片环绕 文字不会被浮动元素压着
